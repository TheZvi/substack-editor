Author Annotations

When copying quotes with Alt+A, matching authors will have their annotation shown in parentheses. Example: theseriousadult (Anthropic): tweet text here...

Website Annotations

When copying quotes from a website, if no author annotation matches, a website annotation will be added. Example: From wsj.com: Some Author (WSJ): quote text here...
